## ğŸ“– Usage
The calculator supports the following operations:

Button	Function

- Number input ('0-9')

- Addition ('+')

- Multiplication ('x')

- Calculate result ('=')

Click operation buttons (+, -, x, /) to select operation

Input the second number

Click "=" to get the result

Click "C" to clear and start fresh
